# STM32 Cortex-M4 labs

Комплект містить вихідні файли для лабораторних робіт 1-3.
Номер залікової книжки: 1.
Варіант лабораторної №2: 1 % 5 = 1.
Варіант лабораторної №3: 1 % 16 = 1.

## Середовище виконання

- arm-none-eabi-gcc
- arm-none-eabi-binutils
- gdb-multiarch або arm-none-eabi-gdb
- make
- qemu-system-gnuarmeclipse з xPack QEMU Arm

## Лабораторна 1

```bash
cd lab1
make
make qemu
```

В іншому терміналі:

```bash
make gdb
```

## Лабораторна 2

Варіант визначається формулою:

```text
номер_залікової % 5 = номер_варіанта
```

Для номера залікової книжки 1 отримуємо варіант 1. Дефолтна збірка вже налаштована на цей варіант:

```bash
cd lab2
make
make qemu
```

У GDB результат перевіряється у регістрі r3.

## Лабораторна 3

Варіант завантажувача визначається формулою:

```text
номер_залікової % 16 = номер_варіанта
```

Для номера залікової книжки 1 отримуємо варіант 1. Дефолтна збірка вже налаштована на LAB2_VARIANT=1 і BOOT_VARIANT=1:

```bash
cd lab3
make
make qemu
```

Звіт і вихідні файли підготовлено для варіантів, визначених за номером залікової книжки 1.
